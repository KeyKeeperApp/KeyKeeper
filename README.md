# KeyKeeper
## Менеджер паролей и ключей

KeyKeeper - личный менеджер паролей и ключей, где пользователь сможет сохранять свои логины, пароли, адреса электронной почты, ключи восстановления с различных веб-сайтов и из приложений и управлять ими.
Также будет возможность настраивать генерацию одноразовых кодов (OTP) для входа на веб-сайты и сохранять ключи симметричного/асимметричного шифрования для личного использования.
Ключи и пароли будут храниться исключительно локально на компьютере пользователя и всё хранилище целиком можно будет защитить мастер-паролем.

## Предзащита

### Главный экран

Здесь Вы можете создать или открыть хранилище, попасть в экран настроек или окно "О приложении".

#### Создание хранилища

После нажатия кнопки создания хранилища на главном экране, Вам будет предложено указать путь, по которому создастся хранилище и создать мастер-пароль. Мастер-пароль нужен для доступа к хранилищу.

#### Открытие хранилища

После нажатия кнопки открытия хранилища на главном экране, Вам будет предложено выбрать хранилище (файл с расширением .kkp) в проводнике. После выбора потребуется ввести мастер-пароль, который был указан при его создании.

### Работа с хранилищем

#### Добавление пароля в хранилище

Для добавления пароля в хранилище нужно нажать кнопку "New Entry", после чего Вам будет предложено ввести название для пароля, а также сам пароль. Также возможна настройка генерации одноразовых паролей. Для этого необходимо указать ссылку на сервис, а также параметры генерации

После добавления пароля нужно нажать кнопку "Save Passwords", которая добавит все новые вхождения в хранилище.

#### Взаимодействие с существующими паролями

Контекстное меню на ПКМ предоставляет следующие возможности:

- Скопировать название пароля

- Скопировать пароль

- Редактировать пароль

- Удалить пароль

Кроме того, пароль вожно редактировать после нажатия кнопки "Edit Selected Entry", если пароль уже выделен.
